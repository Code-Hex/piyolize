## Synopsis

```sh
# argv
$ takopi "はい、いいえ、どうだ"
そうだっピ、ちがうっピ、どうだっピ！
$
# stdin
$ echo "はい、いいえ、どうだ" | takopi
そうだっピ、ちがうっピ、どうだっピ！
```
